## How to Use in Airtable

1. Import `Categories.csv` → creates Categories table
2. Import `Winners.csv` → creates Winners table
3. Import `Submissions.csv` → creates Submissions table (just headers)
4. For each category column in Submissions:
   - Change field type to "Single select"
   - Add the options listed above for that category
5. Create a Form view from the Submissions table
